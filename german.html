Chapter 1: Cases
Cases in german language define how a verb, article, adjectives, etc are conjugated. The german have 4 cases,
  1. : Nominative
  2. : Acusative
  3. : Dative
  4. : Genitive
Note: Genitive is not used in spoken german, only written german.

Nominative

Nominative case is the main actor of a sentence, i.e. Similar to Subject in English. For Example: Der Schauspieler isst den Apfel. Here the Schauspieler(Actor) is in Nominative case. Note: Schauspieler, does infact means actor in german.

Acusative

Accusative case is used for the Main Object of the sentence, i.e. the Main thing the Subject(Nominative) acts upon. For Example: Der Schauspieler isst den Apfel. Here the apfel is eaten by the Actor.

Dative

Dative case is used for secondary object(Only applicable to humans), this one is a bit tricky, So focus here, The Dative case is used for Human/s on whom the Nominative does the Accusative. For Example: Der Schauspieler gebt dich eine Apfel. Here the 'Dich' is in dative. English: The actor gives you the apple. Dich is dative for 'You' and the accusative is 'Eine Apfel'.
